# Frontend - KBeat Store
https://kbeast-store.netlify.app/
Frontend desarrollado con React, Vite, TailwindCSS v4, React Router DOM, Axios, React Hook Form, Toastify y SweetAlert2.

## Instalación

```bash
cd frontend
npm install
npm run dev
```

## Rutas principales

- `/`
- `/login`
- `/register`
- `/products`
- `/products/:id`
- `/cart`
- `/favorites`
- `/orders`
- `/admin/products`
- `/admin/categories`
